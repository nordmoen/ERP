The plugin has been created to allow for morphological change when evolving robot. The morphology of the robot can therefore be encoded using various strategies. The main route for evolving the morphology of the robot is through evolving various compositions of modules to create modular robots. For this, the plugin needs to know what kind of modules should be incoprporated for creating the modular robot. The third tutorial *add hyperlink* demonstrates how the GUI can be used to evolve robots that contain different modules. It also describes how to add your own modules to the system.

It is also possible to evolve a fixed robot morphology using the plugin though it wasn't originally designed for it. For this you can load a fixed robot morphology and evolve a neural network for it. You can choose to optimize using a *add link to feedforward NN* or a *add link to RNN*
